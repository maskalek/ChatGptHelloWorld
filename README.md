# ChatGptHelloWorld

This repository contains a small command-line Sudoku game with a timer and scoring system.

## Running the game

1. Make sure you have Python 3 installed.
2. Run the script:

```bash
python3 sudoku.py
```

Enter your moves in the format `row col value` (for example, `1 3 9`).
The timer starts when the puzzle is displayed. When you complete the puzzle correctly,
your score is calculated based on how quickly you solved it and stored in `scores.json`.
